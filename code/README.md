# `code/` — Claude Code's inbox

Cowork drops errands **here** when a job needs the terminal — files, git, shell, `gh`.
Claude Code sweeps this folder (a `/loop`), acts on `status: open` errands, and marks them `done`.

Named by who *picks work up*, not who drops it. See [`../LEDGER.md`](../LEDGER.md) for what belongs here.
