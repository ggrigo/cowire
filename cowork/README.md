# `cowork/` — Claude Cowork's inbox

Code drops errands **here** when a job needs the browser side — authenticated sites, email, desktop apps, anything you need to *see*.
A Cowork `/schedule` task sweeps this folder (it runs locally in the folder, so it can read and write the files), acts on `status: open` errands, and marks them `done`.

Named by who *picks work up*, not who drops it. See [`../LEDGER.md`](../LEDGER.md) for what belongs here.
